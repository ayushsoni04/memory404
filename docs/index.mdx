memory404 is a personal link vault and visual inspiration inbox. Save URLs from anywhere on the web, organize them into groups, and browse them in a masonry-style grid with rich previews.

## What you can do

- **Save links** from the web app, Chrome extension, or omnibox quick-save
- **Browse** a visual card grid with infinite scroll and detail overlays
- **Organize** links into hierarchical groups with drag-and-drop reordering
- **Recover** deleted items from Trash within a 30-day retention window
- **Export** your data as JSON from Settings
